# habicat
Raspberry Pi based Python project to monitor indoor environmental sensors and correlate with cat activity.

## Components

* Raspberry Pi 4B 2GB
* [Qwiic/STEMMA QT HAT](https://www.adafruit.com/product/4688)

## Sensors

* [BME280 I2C or SPI Temperature Humidity Pressure Sensor](https://www.adafruit.com/product/2652)
* [TSL2591 High Dynamic Range Digital Light Sensor](https://www.adafruit.com/product/1980)
* [SGP40 Air Quality Sensor Breakout - VOC Index](https://www.adafruit.com/product/4829)
